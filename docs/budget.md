## 1. Валюта и курс

Учёт ведём **в двух валютах**: в той, что реально списалась с карты (как есть), и в пересчёте — с **курсом на дату операции** (курс плавает, поэтому фиксируем по каждой строке, а не один на всё).

- Базовая валюта для итогов: **USD** (большинство подписок номинированы в $).
- Справочный курс на 2026-06-18: **≈ 73.6 ₽ / $1** (ЦБ/рынок; см. источники внизу). Для каждой прошлой оплаты ставь курс того дня.
- Если оплата шла через посредника/инокарту с наценкой — в графе «как списано» пиши **фактически списанную** сумму (с наценкой), а не номинал тарифа.

## 4. Будущая операционка (сводка → Track D)

Когда проект поедет в веб (Track D). **Источник правды и детали — [`roadmap.md`](./roadmap.md) → Track D**, здесь только порядок цифр:

- Хостинг приложения (Railway, FastAPI) + БД/auth (Supabase) на бете: **≈ $0–5/мес** (всё на free-tier; Supabase Pro ~$25/мес позже для прод-беты).

- Платёжный процессор: только на Этапе 4 (Freemium); берёт **% с транзакции** (Stripe 2.9%+30¢ / MoR ~5% / крипта ~0.4–2%), фикс-платы нет.
- Итог по Track D: **бета крутится ~бесплатно**, реальные операционные деньги — только когда появятся платящие пользователи.

## 5. Будущие доходы (сводка → icp_and_pricing.md)

**Источник правды и детали — [`icp_and_pricing.md`](./icp_and_pricing.md)**, здесь только модель в двух словах:

- **Freemium.** Free = ядро SRS на встроенных паках + баннер партнёрского рума. Pro ≈ **$8–12/мес или $60–80/год** (гипотеза, ниже конкурентов) = импорт своих диапазонов, аналитика, ad-free, синхронизация.
- Второй поток дохода — **affiliate** партнёрских румов (CoinPoker 30% revshare и т.п.); на ценочувствительном микролимитном сегменте он реалистично основной на раннем этапе.
- Все цифры тарифов — стартовые гипотезы для проверки на бете, не фикс.

## 6. Точка окупаемости (ориентир)

Заполнить, когда будут реальные суммы из §2–§3 и первые платящие.

- **Затраты к окупаемости** = всего потрачено (§3) + накопленная операционка к моменту запуска платежей.
- **Грубый ориентир:** при Pro ≈ $10/мес нужно ≈ `затраты / 10` платящих-месяцев, чтобы выйти в ноль (без учёта affiliate и процентов процессора).
- Это не цель беты — бета про валидацию продукта, не про деньги. Раздел нужен лишь чтобы видеть масштаб.
